# QR Tool + Gemma 3n OCR-WiFi 프로젝트

## 📋 프로젝트 개요
QR 코드 생성/스캔 도구에 Gemma 3n 모델을 활용한 OCR-WiFi 기능 추가

## 🎯 주요 기능
- **QR 생성**: 텍스트/URL → QR 코드
- **WiFi QR**: 수동 입력 + OCR 이미지 추출
- **QR 스캔**: 카메라/이미지 업로드
- **AI OCR**: Gemma 3n으로 WiFi 정보 자동 추출
